CSS 切版：常用佈局速查與範例

這份檔案示範 6 種最常用的切版模式。直接複製下方 HTML 結構與 class，就能快速組裝頁面。

1) 三欄：Header + Sidebar + Content Grid

Header
Sidebar (nav / menu)
Content（文章、表格、列表…）
<div class="layout-3col"> <header class="header">Header</header> <aside class="sidebar">Sidebar</aside> <main class="content">Content</main> </div>

2) Holy Grail 佈局 Header / 內容三欄 / Footer

Header
Left
Center
Right
Footer
<div class="holy-wrap"> <header class="holy-header">Header</header> <div class="holy-main"> <aside>Left</aside> <main>Center</main> <aside>Right</aside> </div> <footer class="holy-footer">Footer</footer> </div>

3) RWD 卡片瀑布流 auto-fill + minmax

Card 1
Card 2
Card 3
Card 4
Card 5
<div class="cards"> <div class="card">...</div> ... </div>

4) 置中技巧 Flex / Grid

Flex 置中
Grid 置中
/* Flex */ .container{display:flex; align-items:center; justify-content:center;} /* Grid */ .container{display:grid; place-items:center;}

5) Sticky Footer Flex

內容區（高度不夠時，Footer 仍貼底）
Footer
/* 版型容器 */ .page{min-height:100dvh; display:flex; flex-direction:column;} .main{flex:1}

6) 等高欄位 Flex 1:1

Column A（等高）
Column B（等高）
Column C（等高）
.container{display:flex; gap:12px} .container > div{flex:1} @media(max-width:780px){.container{flex-direction:column}}

小抄： Grid 三寶：repeat / auto-fit(auto-fill) / minmax Flex 三寶：gap / align-items / justify-content RWD：@media + clamp() + fluid spacing
